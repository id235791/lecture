DML(Data Manipulation Language)
	데이터 조작어
	실질적으로 데이터들을 CRUD하는 언어
	
	데이터 추가
		INSERT INTO 테이블명 [ (컬럼명1,컬럼명2, ..) ]
		VALUES(값1,값2,...);

	데이터 검색
		SELECT 컬럼명1,컬럼명2,...
		FROM 테이블명 WHERE 조건식;

	데이터 수정
		UPDATE 테이블명
		SET 컬럼명1 = 바꿀값1, 컬럼명2 = 바꿀값2, ...
		WHERE 조건식;

	데이터 삭제
		DELETE FROM 테이블명 WHERE 조건식;
		
조건절
	=	: 같다
	!=, <>	: 다르다
	AND	: 두 조건식이 모두 참이면 참
	OR	: 두 조건식중 하나라도 참이면 참

정규화
	삽입, 수정, 삭제의 이상현상을 제거하기 위함
	데이터 중복을 최소화 하기 위함
	대부분 3차 정규화까지 진행
