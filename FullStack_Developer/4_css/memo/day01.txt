CSS(Cascading Style Sheets)
	HTML 문서를 시각적으로 꾸며주는 역할을 하는 언어
	요소가 구체적으로 어떤 스타일로 표현되는지를 정의하는 규격
	HTML 내부적으로 요소의 디자인을 할 수 있지만,
	가독성이 떨어지고 유지보수가 좋지 않으므로 해결하기 위해
	css를 사용한다.(내용과 스타일을 분리하여 분업을 한다.)

CSS 문법
	선택자 {
		속성명:속성값;
		속성명:속성값;
		...
	}

	p { color:red; text-align:center; }
	-- -------------------------------
	선택자		선언부

	HTML 문서 중 디자인을 적용시킬 태그 안에 style 속성에 작성
	HTML 문서 <head> 태그 안에 <style> 태그 안에 css 문법을 작성
	.css 파일을 만들어서 그 안에 css 문법을 작성

CSS를 적용하는 방법
	1. 인라인 스타일
		해당 요소에만 스타일을 적용(선택자가 필요없다)
		<p style="color:red;">안녕하세요</p>

	2. 내부 스타일 시트
		해당 문서에서만 스타일을 적용(선택자가 꼭 필요함)

		<head>
		<style>
			css 문법
		</style>
		</head>

	3. 외부 스타일 시트
		웹 사이트 전체의 스타일을 하나의 파일에서 변경, 관리
		.css 파일을 만들어서 css문법을 작성하고 디자인을 적용시킬 곳마다
		<head> 태그 안에 <link> 태그를 사용해서 연결, 적용하는 방법

		<link rel="stylesheet" href="css파일경로">
		rel 속성은 현재 파일과 링크된 파일 사이의 연관관계를 명시

CSS 선택자
	1. 전체 선택자
		스타일을 모든 요소에 적용(초기화용 코드에 많이 사용)
		디테일한 스타일은 전체 선택자나 body 태그에 적용한 스타일보다
		다른것이 더 우선시 됨

		* {
			css 문법
		}

	2. 태그 선택자, 요소 선택자
		특정 태그가 쓰인 모든 요소에 스타일을 적용

		태그이름 {
			css 문법
		}

		상속 : 부모 요소의 속성값이 자식 요소에게 적용되는 성질
		https://www.w3.org/TR/CSS22/propidx.html

	3. id 선택자
		웹 문서 안의 특정한 요소, 부분에 스타일을 적용
		html 요소의 id 속성값을 이용
		# 기호를 이용

		#아이디명 {
			css 문법
		}

	4. class 선택자
		웹 문서 안의 특정 집단의 요소에 한번에 스타일을 적용
		html 요소의 class 속성값을 이용
		. 기호를 이용

		.클래스명 {
			css 문법
		}

	5. 그룹 선택자
		여러개의 선택자를 나열하고 ,로 구분해 스타일을 적용

		선택자1, 선택자2 {
			css 문법
		}

	6. 자손 선택자
		조상요소 하위에 있는 자손을 모두 선택해 스타일을 적용
		자손은 자식을 포함

		조상요소선택자 자손요소선택자 {
			css 문법
		}

	7. 자식 선택자
		부모 요소 하위에 있는 자식을 선택해 스타일을 적용

		조상요소선택자 > 자식요소선택자 {
			css 문법
		}

	8. 인접 형제 선택자
		형제 관계에서 연속적으로 존재하는 요소를 선택할 때 사용

		형선택자 + 동생선택자 {
			css 문법
		}

	9. 일반 형제 선택자
		형제 관계로 있는 요소들을 선택할 때 사용
		형 요소 다음에 오는 모든 동생 요소들을 선택

		형선택자 ~ 동생선택자 {
			css 문법
		}

	10. 속성 선택자
		HTML 요소에는 src, href, style, type, ... 과 같은 속성을
		선택자로 지정해서 스타일을 적용

		[속성명]{
			css 문법
		}

		[속성명]		: 속성을 가지고 있는 모든 요소들 선택
		[속성명=속성값]	: 속성의 값이 지정한 값인 요소들 선택
		[속성명~=값]	: 속성값에 "값"이 독립적으로 존재하고 있는 요소를 선택
				완벽히 독립적으로 해당 "값"이 존재해야 함
				~="apple" --> "apple banana" 인것도 선택
		[속성명*=값]	: "값"이 속성값에 포함되어 있는 요소를 선택
		[속성명$=값]	: "값"이 접미사로 끝나는 속성값을 가지고 있는 요소를 선택
		[속성명|=값]	: "값"이 접두사로 시작하는 속성값을 가지고 있는 요소를 선택
				"값"만 속성값으로 존재하거나, 하이픈으로 연결된 것만 선택
				|="apple" --> "apple", "apple-mango"
		[속성명^=값]	: "값"이 접두사로 시작하는 속성값을 가지고 있는 요소를 선택
				공백 및 언더바로 연결된 것도 선택
				^="apple" --> "apple", "apple mango", "apple_mango"
