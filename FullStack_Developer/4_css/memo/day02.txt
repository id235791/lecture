CSS 선택자
	11. 순서에 따른 가상 클래스 선택자
		클래스를 추가할 필요 없이 요소 중에서 순서에 따라
		원하는 특정 요소를 선택하여 스타일을 지정
		:first-child / :last-child / :nth-child() / ...

	12. 상태에 따른 가상 클래스 선택자
		:hover
		:link
		:visited
		:active
		:checked
		...

	13. 가상 요소 선택자
		가상요소는 페이지 내에 실제로 존재하는 요소가 아니기 때문에
		자바스크립트로 컨트롤이 불가능하다. 따라서 단순히 디자인적으로만
		활용해야 할 때 사용을 한다.

		선택자::after
			선택자 이후에 가상의 요소가 있다고 가정한 후 스타일을 지정
			글, 이미지, 그라데이션 등을 선택자 뒤에 삽입
			
		선택자::before
			선택자 이전에 가상의 요소가 있다고 가정한 후 스타일을 지정
			글, 이미지, 그라데이션 등을 선택자 앞에 삽입

CSS의 컬러
	색상 이름으로 표현
		red, white, deepskyblue, ...

	16진수 색상값으로 표현
		rgb 색상값을 16진수로 표현
		
		0~255	0~255	0~255
		00~FF	00~FF	00~FF

		#000000 ~ #FFFFFF
		검정	~	흰색

	RGB 색상값으로 표현
		검정	~	흰색
		rgb(0,0,0)	rgb(255,255,255)
		rgba(0~255,0~255,0~255,0 ~ 1)
					----
				alpha 투명도(높을수록 불투명)

CSS의 텍스트
	color
		텍스트의 색상을 설정(기본색은 검정색)
	letter-spacing
		텍스트 내에서 글자 사이의 간격을 설정
	word-spacing
		텍스트 내에서 단어 사이의 간격을 설정
	text-align
		텍스트의 수평 방향 정렬을 설정
	text-indent
		단락의 첫 줄의 들여쓰기(내어쓰기)를 설정
	line-height
		텍스트의 줄 사이의 간격(높이)을 설정
	text-decoration
		텍스트에 가로줄을 이용하여 여러가지 효과를 설정하거나 제거하는데 사용
	text-transform
		텍스트에 포함된 영문자에 대한 대소문자를 설정
	font-variant
		소문자를 같은 크기의 대문자로 변경
		소문자 크기의 작은 대문자로 설정
	text-shadow
		텍스트의 그림자 효과를 설정
		text-shadow:가로거리 세로거리 번짐정도 색상;
	white-space
		스페이스와 탭, 줄바꿈, 자동 줄바꿈을 어떻게 처리할지 설정
	overflow
		기준선을 벗어나 넘치는 경우 content를 어떻게 처리할지 설정
		내부 내용(content)가 요소의 크기를 벗어났을 때 처리 방법을 설정
	text-overflow
		텍스트가 요소를 벗어나 넘친 경우 넘치는 텍스트의 표현을 어떻게 할지 설정
	font-size
		텍스트의 크기를 설정
		px
			스크린의 픽셀을 기준으로 하는 절대 크기 설정
		%
			기본크기(16px)를 기준으로 하고 상대 크기 설정
		em
			부모 요소로부터 상속받는 글자 크기를 기준으로 하고 상대 크기 설정
		rem
			부모 요소에 상관없이 html 태그의 글자 크기에 대한 상대 크기 설정
	font-weight
		텍스트의 굵기를 설정
		기본 굵기는 숫자 400 / normal
		두껍게 700 / bold
		숫자는 100 ~ 900 사용
	font-family
		글꼴을 설정
		여러개의 글꼴을 등록한 후 그 중 하나의 글꼴을 설정
		여러개의 글꼴이 등록되어 있는 경우, 웹 브라우저에서 앞에 있는 것부터 순서대로
		사용 가능 여부를 판단한 뒤 적용
		글꼴 이름에 띄어쓰기가 포함되어 있는 경우 반드시 따옴표로 감싸주어야 함
