플렉서블 레이아웃(flex)
	대부분의 웹 사이트는 수직 구성
	레이아웃을 구성할 때 가장 많이 사용하는 요소들은 대부분 블록 요소
	수직 구성은 쉽게 구성할 수 있으나, 수평 구성은 구현하기 쉽지 않음
	과거에는 수평 구성을 <table>, float, inline-block 으로 구성
	좀 더 편하고 쉽게 수평구성을 구현하기 위해서 flex를 사용한다.

flex-wrap
	플렉스 라인에 더 이상 여유가 없을 때 플렉스 요소의 위치를 어떻게 할지를 설정

	nowrap : 기본설정. 플렉스 요소가 다음줄로 넘어가지 않음. 요소의 너비를 줄여 한 줄에 모두 배치
	wrap : 플렉스 요소가 여유 공간이 없으면 다음 줄로 넘어감.
	wrap-reverse : 플렉스 요소가 여유 공간이 없으면 이전 줄로 넘어감(위쪽으로 넘어감)

justify-content
	플렉스 요소의 수평방향 정렬 방식을 설정

	flex-start : 기본설정, 앞쪽에서부터 배치됨
	flex-end : 뒤쪽에서부터 배치됨
	center : 가운데에 배치됨
	space-between : 요소들 사이에 여유 공간을 두고 배치됨
	space-around : 앞, 뒤 그리고 요소들 사이에 모두 여유 공간을 두고 배치됨
	space-evenly : 앞, 뒤 그리고 요소들 사이에 공간을 모두 일정하게 두고 배치됨

align-items
	플렉스 요소의 수직방향 정렬 방식을 설정

	stretch : 기본설정. 플렉스 요소의 높이가 컨테이너의 높이와 같게 변경된 뒤 연이어서 배치됨
	flex-start : 플렉스 요소는 플렉스 컨테이너의 위쪽에 배치됨(기준선은 위쪽 선)
	flex-end : 플렉스 요소는 플렉스 컨테이너의 아래쪽에 배치됨(기준선은 아래쪽 선)
	center : 플렉스 요소는 플렉스 컨테이너의 가운데에 배치됨
	baseline : 플렉스 요소는 플렉스 컨테이너의 기준선에 배치됨(내용으로 기준선을 잡음)

align-self
	플렉스 요소마다 서로 다른 align 속성값을 설정할 수 있다.

order
	플렉스 요소들의 순서를 설정

flex-grow
	플렉스 아이템들이 차지할 너비에 대한 증가형 숫자를 지정

flex-shrink
	플렉스 아이템들이 차지할 너비에 대한 감소형 숫자를 지정

flex-basis
	플렉스 아이템들의 길이를 지정

flex
	flex의 속성을 한꺼번에 설정
	flex:1 1 0;
	grow shrink basis

미디어 쿼리
	width, height, color, .. 속성과 같은 미디어 관련 표현식을 통해
	스타일이 적용되는 범위를 조절하는 기법
	반응형 개발
	컨텐츠를 별도로 변경하지 않아도 웹 페이지에 접속하고 있는 기기에 따라
	알맞은 형태로 스타일을 조정

	장점 : UI 고급화, 유지관리와 업데이트
	단점 : 다운로드 속도가 느리다, 구현이 어렵다.

	기본 스타일링 코드
	<style>
		선택자{
			속성명:속성값;
		}

	</style>

	미디어 쿼리
	<style>
		@media 매체유형 and 조건식 ... {
			선택자{
				속성명:속성값;
			}
		}
	</style>

	매체 유형
		all	: 모든 매체
		screen	: 컴퓨터, 태블릿, 스마트폰, ...
		print	: 프린터 기기
		speech	: 스크린 리더기
