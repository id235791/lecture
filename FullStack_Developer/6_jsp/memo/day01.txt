웹 컨테이너(서블릿 컨테이너)
	동적인 데이터 연산, 요청이 들어왔을 때 웹 서버가 연산을 요청하는 곳
	연산이 끝나면 정제된 데이터(html, json, xml, jpeg, ...)로 서버에 돌려준다.

아파치 톰캣(WAS) 설치
	톰캣 9.0버전 사용
	구글에 tomcat 검색 > 홈페이지 입장 > 왼쪽 다운로드에서 tomcat 9 클릭
	> 64-bit Windows zip 클릭 > 압축 해제

서블릿(Servlet)
	JAVA 코드로 HTML 문서를 생성, 작성할 수 있는 JAVA 프로그램이다.
	ex) out.print("<html>");
	클라이언트의 요청을 받고 데이터를 처리하는 방법들을 제공한다.

JSP(Java Server Page)
	HTML을 중심으로 자바와 같이 연동하여 사용하는 웹 언어이다.
	HTML 코드 안에 특정 태그를 작성하면 JAVA 코드를 그 내부에 쓸 수 있다.

스크립트 태그
	HTML 문서 내부에 자바 코드를 넣어 프로그래밍이 가능하도록 만들 수 있다.

	1. 선언문(Declaration)
		<%! %>
		자바 변수나 메소드를 정의하는데 사용되는 태그
		변환되는 서블릿 클래스 영역에 선언된다.

	2. 표현문(Expression)
		<%= %>
		자바 변수, 계산식, 메소드 호출의 리턴 값 등을 표현해주는 태그
		작성한 값이 그대로 HTML 문서에 표현된다.
		표현된 때의 타입은 문자열이다.

	3. 스크립틀릿(Scriptlet)
		<% %>
		자바 변수 선언 및 자바 로직 코드를 작성하는데 사용되는 태그
		변환되는 서블릿 클래스에서 응답을 만들어내는 메소드 내부에
		그대로 코드가 옮겨진다.
